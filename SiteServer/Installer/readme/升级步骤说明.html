升级步骤说明

          升级步骤说明请参考官方网站《原有系统升级步骤》

    特别注意：

    1、如果原来版本是SiteServer CMS 4.0（含）以上的且在IIS中已经把API转换成为了应用程序的则需要把此API应用程序删除，否则系统不能正常运行；

    2、查看IIS应用程序池，必须采用集成模式；

    3、查看.Net Framework，必须是.Net Framework 4.5.1以上版本；

    4、如果安装了SiteServer Service 服务组件，升级之前，需要进入服务器暂时关闭组件，否则会影响升级。
