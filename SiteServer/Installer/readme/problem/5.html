访问安装向导时提示：Service Unavailable


问题原因

    有很多原因可能导致此问题，最可能的原因是.NET版本冲突

解决方法

    在IIS中可以设置网站属性，同时网站下面可以添加虚拟目录。出现Service Unavailable的情况很多时候是由于IIS中网站属性中配置使用的.NET版本与虚拟目录中配置使用的.NET版本不同。将虚拟目录中属性的.NET版本与网站属性中.NET版本统一即可解决此问题。

    如果首先安装.NET后安装IIS也可能导致此问题，正确的顺序是先安装IIS后安装.NET。如果是安装顺序导致的问题，重新下载.NET并安装能够解决。

    除此之外，.NET补丁要及时更新，确保.NET框架的稳定。
