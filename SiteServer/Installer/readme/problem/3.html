访问网站时出现 SiteServer 系列产品登录页


问题原因

    系统根目录下的“default.aspx”默认页将自动转向到 SiteServer 系列产品管理后台界面。

解决方法

    系统根目录下默认存在“default.aspx”文件，用于将页面转向到 SiteServer 系列产品管理后台，便于快速进行后台页面。

    当网站正式上线后，需要删除根目录下的“default.aspx”页面，以显示网站内容。
