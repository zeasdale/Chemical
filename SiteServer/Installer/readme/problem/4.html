运行安装向导时提示: 用户 'sa' 登录失败


问题原因

    您所使用的SQL Server帐号权限不足，或者SQL Server拒绝远程连接。

解决方法

    检查您SQL Server帐号的权限，如果您的web服务器和SQL Server在同一台服务器上我们推荐您将数据库地址设置为“(local)”这样可以解决大部分无法连接SQL Server的问题。部分Windows Server 2003的用户SQL server2000拒绝链接可能由于没有及时安装SQL Server 2000的SP补丁引起的。建议您及时安装补丁。
